# Social Post Templates

## Usage
When user asks for "social posts", "write posts", or "caption ideas", generate platform-specific content.

---

## Instagram Post Structure

### Format
```
[HOOK - Line 1 - Big statement or question]
[LINE 2 - Setup or context]
[LINE 3 - Value or story]
[LINE 4 - CTA or engagement prompt]

.

[Hashtags - 8-15]
```

### Example - Tattoo Shop Reopening
```
WE'RE BACK 🔥

After 3 months of healing (and some questionable quarantine haircuts)
our doors are finally open again.

Same artists, same passion, same clean needles.
Book your consultation link in bio.

.

#tattoos #tattooartist #newtattoo #tattooed #tattooshop
#fortworth #dfwtattoo #tattoolove #ink #bodyart
#reopening #backtobusiness
```

---

## Facebook Post Structure

### Format
```
[Attention-grabbing headline]

[Paragraph 1 - The hook and context]

[Paragraph 2 - Details, benefits, story]

[CTA - What you want them to do]

#[hashtags - fewer than Instagram]
```

---

## TikTok Video Structure

### Hook Types
- "POV: You're getting a tattoo and..."
- "3 things I wish I knew before my first tattoo"
- "Watch till the end - you won't believe how this turned out"
- "Storytime: The weirdest request I ever got..."

### Caption Format
```
[Hook question or statement]
.
[Context]
.
[FYP #fyp #foryou #viral]
```

## Engagement Post Templates

### Question Post
```
What's something you've always wanted to do but been too scared to try?

For me, it was [personal story - vulnerability builds connection]

Drop a 👇 if you relate 👇
##[hashtag] ##[hashtag]
```

### This or That
```
BLACK TATTOO ART or COLOR? 🎨⚫

Let us know below! ⬇️⬇️
##tattoos ##tattooartist ##blackwork
```

### Behind the Scenes
```
You asked, we answered 🙌

Here's what really happens during a [service]...

[3-4 process bullet points]

Tag someone who needs to see this 👇
##behindthescenes ##tattooprocess
```

## Promotional Post Templates

### Standard Promo
```
[HEADLINE - Benefit focused]

[3 bullet points of what they're getting]

🔥 LIMITED TIME: [Offer details]

👉 [Link in bio / Button link]

Hurry, [deadline or scarcity]
```

### Grand Opening
```
IT'S HAPPENING 🎉

[Business Name] is officially OPEN!

📍 [Address]
📞 [Contact]
🔗 [Link in bio]

First 20 customers get [incentive]!
Come say hi 👋
##grandopening ##newbusiness
```
