# Marketing Agent Skills

## Overview
OpenClaw skills for Kali's marketing agent. Designed for a MacBook host with Canva, email, and cloud access.

| Skill | Purpose | Status |
|-------|---------|--------|

| **canva-connect** | Generate designs via Canva API | 🔜 Next |
| **email-nurture** | Build and manage email sequences | 🔜 Next |
| **lead-tracker** | Track leads through pipeline stages | 🔜 Next |

```

Agent: Builds full campaign plan, email sequence, social calendar
```

### Generate Content
```
User: Write 5 social posts for our tattoo shop reopening
Agent: Creates engaging posts with hashtags, timing suggestions
```

### Track Leads
```
User: Add new leads from today's event
Agent: Creates reminder tasks, follow-up sequence
```
